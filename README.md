# simple-web-react

シンプルなWebUIの実装サンプル（React）

>言語はJavaScript利用。

## ブランチ情報

- main
  - シンプルなWebAPIと連携する基本実装


## 開発・動作確認環境

- macOS(Apple Silico)
- Node.js（24.x）
- Google Chrome

## 利用方法

```bash
git clone https://github.com/bluecode-jp/simple-web-react.git
cd simple-web-react
npm install
npm run dev
```
